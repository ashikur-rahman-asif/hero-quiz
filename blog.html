1.Difference between map,filter,foreach and find

In JavaScript, map, filter, forEach, and find are all methods that can be called on arrays to manipulate or iterate over them.

=>Map is a method that creates a new array by calling a provided function on every element in the original array. The provided function takes the current element, its index, and the original array as arguments, and should return a new value to be included in the new array.
=>Filter is a method that creates a new array with all elements that pass the test implemented by the provided function. The provided function takes the current element, its index, and the original array as arguments, and should return a boolean value indicating whether the element should be included in the new array.
=>ForEach is a method that calls a provided function on every element in the array, but does not create a new array. The provided function takes the current element, its index, and the original array as arguments, but does not return anything.
=>Find is a method that returns the value of the first element in the array that satisfies the provided testing function. The provided function takes the current element, its index, and the original array as arguments, and should return a boolean value indicating whether the element satisfies the search criteria.
